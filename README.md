# The Swarm — הוראות הפעלה וכללי משחק

איך להריץ (IntelliJ IDEA):
- פתחו את התיקייה כ-Project ב-IntelliJ.
- ודאו ש-`src` מסומן כ-Source Root (אם צריך).
- הריצו את מחלקת `Main` (Run -> Run 'Main').
- IntelliJ בדרך כלל יטפל במשאבים, אך אם לא, וודאו ש-`res` נמצא בתצורת ה-Resources או הוסיפו אותו ל-classpath.

בקרות (Controls):
- W A S D — להזיז את השחקן (Up / Left / Down / Right).
- SPACE — התקפה (Slash). יש קירור בין התקפות.
- ESC — הפסקה (Pause) — מצב הפעלה/השבתה.
- R — בעת מצב "GAME OVER" לחיצה על R תאתחל מחדש את המשחק.

מטרת המשחק וחוקים (בקצרה):
- המטרה: להישאר בחיים כמה שיותר ולצבור נקודות על ידי השמדת האויבים.
- לכל אויב שהשמדתם תקבלו +100 נקודות.
- נקודות מגדירות את הרמה: הרמה עולה ככל שהצטברות הנקודות גדלה; רמת המשחק משפיעה על תדירות ההופעה של האויבים.
- לשחקן יש 3 נקודות חיים (HP). מגע עם אויב מוריד חיים ב-1.
- כאשר ה-HP מגיע ל-0 המשחק נגמר ("GAME OVER"). ניתן ללחוץ `R` כדי להתחיל מחדש.
- יש חץ זמן קצר של "hit cooldown" אחרי שנפגעים שבו לא ניתן להיפגע שוב מיד.
- יש קירור בין התקפות כך שלא ניתן להחזיק את מקש ה-Space ולתקוף ללא הפסקה.

טיפים נחמדים:
- כוונו את הפנים של השחקן לפני ההתקפה כדי להכות אויבים שמופיעים בצד הרצוי.
- ככל שהרמה עולה, שיעור ההופעה של האויבים גדל — שימו לב לכך.

קבצים חשובים בפרויקט:
- `src\Main.java` — נקודת הכניסה של התוכנית.
- `src\WindowFrame.java` — יצירת החלון ותפריט התחלתי.
- `src\GamePanel.java` / `src\GameEngine.java` — הלוגיקה והציור של המשחק.
- `res/` — תמונות וקבצי סאונד (player.png, enemy.png, background.png, slash.wav וכו').

בהצלחה ותהנו!
